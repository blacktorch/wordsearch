# WordSerach

<p>This is a Word Search `Android` Application written in `Java`. It has a 10 X 10 grid that contains the character squence
of these following words:</p>

<ol>
<li>JAVA</li>
<li>SWIFT</li>
<li>KOTLIN</li>
<li>MOBILE</li>
<li>VARIABLE</li>
<li>OBJECTIVEC</li>
</ol>

<p>The characters of these words, are randomized across the grid (Multi-Dimensional Array) each time the application is refreshed.</p>

### UI/UX ###

<p>It has an intuitive user interface that allows the user to swipe across the grid and find words.
There are various animations to give the user an immersive experience, and also haptic feedback when words are found.</p>
